INSTRUCCIONES DE USO - QUIZ DE PRODUCTIVIDAD Y TIEMPOS

Descripción: Esta aplicación es un Quiz interactivo desarrollado en
Streamlit cuyo objetivo es evaluar conocimientos básicos sobre
productividad, tiempos y eficiencia en procesos industriales.

Enlace a la aplicación:
https://quiz-industrial-equipo4-tjksnuo7srirvj3suhkunj.streamlit.app/

Cómo utilizar la aplicación:

1.  Ingreso al Quiz:
    -   Abre el enlace proporcionado en tu navegador.
    -   La aplicación cargará la página principal titulada “Quiz de
        Productividad y Tiempos”.
2.  Registro de participante:
    -   En la pantalla inicial, escribe tu nombre completo en el campo
        “Ingresa tu nombre”.
    -   Presiona el botón “Comenzar Quiz”.
    -   Si no ingresas un nombre, la aplicación te pedirá que lo hagas
        antes de continuar.
3.  Responder las preguntas:
    -   Una vez registrado, se mostrarán 10 preguntas de opción
        múltiple.
    -   Para cada pregunta, selecciona la respuesta que consideres
        correcta haciendo clic en una de las opciones.
    -   Puedes cambiar tu respuesta en cualquier momento antes de
        finalizar el quiz.
4.  Finalizar el Quiz:
    -   Cuando hayas respondido todas las preguntas, presiona el botón
        “Finalizar Quiz”.
    -   La aplicación evaluará automáticamente tus respuestas.
5.  Ver resultados:
    -   Se mostrará tu puntaje total obtenido de 10.
    -   También se mostrará tu calificación final en una escala de 0 a
        10.
6.  Reiniciar el Quiz:
    -   Si deseas realizar nuevamente el quiz, presiona el botón
        “Reiniciar Quiz”.
    -   Esto borrará tus respuestas y te pedirá nuevamente tu nombre
        para comenzar de nuevo.

Notas importantes: - El orden de las preguntas y las opciones se genera
de forma aleatoria en cada intento. - El quiz está diseñado para fines
de capacitación y evaluación de conocimientos básicos en productividad
industrial.
